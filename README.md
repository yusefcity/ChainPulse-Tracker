# ChainPulse-Tracker
Simulation system designed to model simplified blockchain-style transaction flows. It demonstrates how Blockchain structures can be represented using clean in-memory logic without external dependencies. The system also simulates basic Cryptocurrency movement between virtual accounts to illustrate how distributed-style balances and transaction records can be managed.

The project is written as a single TypeScript file to emphasize clarity and architectural readability. It focuses on core programming concepts such as data modeling, class design, and state tracking. Developers can use this project as a learning reference for understanding how transaction chains and ledger updates can be structured in a real application.

ChainPulse Tracker is intentionally lightweight but structured in a way that mirrors real-world backend design patterns.
